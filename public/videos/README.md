# Video Assets

Place your talking-head video as `talking-head.mp4` in this directory.

The video will be used as the main background asset for the cinematic hero section.

**Supported formats:** MP4 (H.264 recommended for best browser compatibility)

**Recommended specifications:**
- Resolution: 1920x1080 or higher
- Duration: 10-30 seconds
- File size: Keep under 50MB for web performance
- Codec: H.264 for maximum browser support
